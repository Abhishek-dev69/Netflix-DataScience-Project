# Netflix Data Science Project

## Project Overview
This project analyzes Netflix movie data to discover insights such as genre distribution, ratings, and trends.

## Dataset
- Source: Movie Database
- File: mymoviedb.xlsx

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Analysis
- Data cleaning
- Exploratory Data Analysis (EDA)
- Visualizations
- Insights and conclusions

## How to Run
1. Clone the repository
2. Open the Jupyter Notebook
3. Run all cells
